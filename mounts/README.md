# mounts/

3D-печатные детали, не входящие в оригинальную модель NITROUS (автор не
занимался автономным вождением по камере — см. `EXTERNAL_SOURCES.md`,
раздел про TommyB).

## camera_mount.scad

Параметрическое крепление камеры Raspberry Pi Camera Module 3 на шасси.
Написано на OpenSCAD (текстовый CAD — сам файл является кодом, изменение
размеров — это правка чисел в начале файла, не работа руками в GUI).

### Источник размеров

Официальный механический чертёж Raspberry Pi:
https://datasheets.raspberrypi.com/camera/camera-module-3-standard-mechanical-drawing.pdf

Оттуда взяты:
- Габарит платы камеры: 25 × 23.862 мм
- Диаметр монтажных отверстий: 2.2 мм
- Примерный шаг между отверстиями: ~21.6 × ~20.8 мм (считано с чертежа)

### Почему пазы, а не точные отверстия

Считывание точного шага отверстий с технического PDF без камеры в руках
несёт риск ошибки в 1-2 мм. Вместо точных круглых отверстий под этот шаг
крепление использует **вытянутые пазы** (`hull()` из двух цилиндров) —
это даёт запас на неточность и позволяет крепежу совпасть, даже если
реальная плата слегка отличается от чертежа (другая ревизия, погрешность
чтения).

### Статус: НЕ скомпилировано

В этом окружении нет установленного OpenSCAD (первая попытка через
Chocolatey упала на правах администратора; запущена вторая попытка через
winget в фоне — см. `README.md`, roadmap). Файл написан аккуратно, только
базовыми примитивами (`cube`, `cylinder`, `translate`, `rotate`,
`difference`, `union`, `hull`), но **перед печатью обязательно открыть в
OpenSCAD, визуально проверить и, по возможности, распечатать пробную
версию только головной пластины** (без стойки) для проверки посадки
камеры перед печатью всей детали.

### Параметры для правки (в начале файла)

| Параметр | Что регулирует |
|---|---|
| `post_height` | Высота стойки — зависит от того, куда крепится на шасси |
| `head_tilt_deg` | Наклон камеры вниз к трассе — подбирается по факту после установки, см. `tools/calibrate_camera.py` |
| `base_x`, `base_y` | Размер опорной площадки под шасси |
| `cam_hole_pitch_x/y` | Шаг монтажных отверстий камеры — **перепроверить штангенциркулем по реальной плате перед печатью** |

### Как использовать

```bash
openscad -o camera_mount.stl camera_mount.scad
```

Открыть `.stl` в слайсере, проверить масштаб/ориентацию, напечатать.
